# 生产者-消费者共享内存项目

这是一个完整的生产者-消费者模型实现，使用System V共享内存和信号量来实现进程间通信。该项目展示了如何在实际应用中使用共享内存进行高效的数据交换。

## 项目概述

### 功能特性
- **多生产者多消费者**: 支持多个生产者和消费者同时运行
- **环形缓冲区**: 使用环形缓冲区实现高效的数据存储和检索
- **同步机制**: 使用多个信号量确保数据一致性
- **实时监控**: 提供监控程序实时显示系统状态
- **可视化界面**: 监控程序提供缓冲区状态的可视化显示

### 技术特点
- 使用System V共享内存实现进程间数据共享
- 使用三个信号量实现同步控制
- 支持动态配置生产/消费速度
- 提供详细的运行状态和统计信息

## 核心概念

### 1. 环形缓冲区
```c
typedef struct {
    product_t buffer[BUFFER_SIZE]; // 环形缓冲区
    int in;                       // 写入位置
    int out;                      // 读取位置
    int count;                    // 当前产品数量
    int total_produced;           // 总生产数量
    int total_consumed;           // 总消费数量
    time_t start_time;            // 开始时间
} shared_buffer_t;
```

### 2. 产品结构
```c
typedef struct {
    int id;                     // 产品ID
    time_t timestamp;           // 生产时间戳
    pid_t producer_pid;         // 生产者PID
    char data[MAX_MESSAGE_LEN]; // 产品数据
} product_t;
```

### 3. 同步机制
项目使用三个信号量来实现同步：

- **互斥信号量 (mutex)**: 保护共享缓冲区的互斥访问
- **空槽位信号量 (empty)**: 表示可用空槽位数量，初始值为BUFFER_SIZE
- **满槽位信号量 (full)**: 表示可用产品数量，初始值为0

## 编译和运行

### 1. 编译程序
```bash
cd 07_projects
make all
```

### 2. 基本运行
需要打开三个终端：

**终端1 - 启动生产者**:
```bash
./producer 1 20 1000000
```
参数说明：
- `1`: 生产者ID
- `20`: 生产20个产品
- `1000000`: 生产间隔（微秒，1秒）

**终端2 - 启动消费者**:
```bash
./consumer 1 20 1500000
```
参数说明：
- `1`: 消费者ID
- `20`: 消费20个产品
- `1500000`: 消费间隔（微秒，1.5秒）

**终端3 - 启动监控**:
```bash
./monitor -i 2
```
参数说明：
- `-i 2`: 每2秒更新一次状态

### 3. 使用Makefile快捷命令

#### 演示模式
```bash
make demo
```
自动启动多个终端进行演示。

#### 压力测试

启动多个生产者和消费者进行压力测试。

#### 性能测试

## 相关概念

### 1. 生产者-消费者问题
经典的同步问题，需要解决：
- 互斥访问共享缓冲区
- 缓冲区满时生产者等待
- 缓冲区空时消费者等待

### 2. 信号量同步
- **P操作 (sem_wait)**: 获取资源
- **V操作 (sem_post)**: 释放资源
- **互斥信号量**: 保证互斥访问
- **计数信号量**: 控制资源数量

### 3. System V IPC
- **共享内存**: 高效的进程间数据共享
- **信号量**: 进程间同步机制
- **消息队列**: 进程间消息传递

## 性能指标

### 1. 吞吐量
- 生产速率: 每秒生产的产品数
- 消费速率: 每秒消费的产品数
- 系统吞吐量: 每秒处理的请求数

### 2. 延迟
- 生产延迟: 从开始生产到放入缓冲区的时间
- 消费延迟: 从放入缓冲区到被消费的时间
- 端到端延迟: 从生产到消费的总时间

### 3. 资源利用率
- 缓冲区利用率: 实际使用容量/总容量
- CPU利用率: 进程CPU使用率
- 内存利用率: 共享内存使用效率
